Utilización práctica

Uso del sistema IoT por parte del personal
Objetivo del sistema
El objetivo del sistema IoT es facilitar el trabajo diario del personal de Letras del Norte, ayudando a localizar los libros rápidamente, mantener el inventario actualizado y mejorar la atención al cliente, sin necesidad de conocimientos técnicos avanzados.

Cómo utilizar el sistema en el día a día
Cada libro dispone de un pequeño localizador que permite conocer su ubicación exacta dentro de la librería.
El personal puede consultar el sistema desde un ordenador, tablet o dispositivo móvil autorizado.
A través de una aplicación sencilla, se puede buscar un libro por título, autor o código, y el sistema mostrará dónde se encuentra y cuántas unidades hay disponibles.

Gestión de pedidos y reservas
Cuando un cliente realiza un pedido o una reserva, el sistema indica automáticamente qué libros están disponibles.
El trabajador solo tiene que seguir la ubicación indicada para preparar el pedido de forma rápida y sin errores.
Una vez entregado el libro, el stock se actualiza automáticamente, sin necesidad de realizar anotaciones manuales.

Uso del sistema durante eventos
Antes del evento, el personal puede comprobar qué libros estarán disponibles para la venta o firma.
Durante el evento, el sistema permite saber en tiempo real cuántos libros quedan, evitando confusiones o ventas duplicadas.
Esto ayuda a mantener una mejor organización y a ofrecer una experiencia más fluida a los asistentes.

Ventajas para el personal
Menos tiempo dedicado a tareas repetitivas de control de inventario.
Reducción de errores y estrés en momentos de alta afluencia.
Mayor facilidad para atender a los clientes de forma rápida y eficaz.
